# SmartAccess IoT

Projeto de controle de acesso utilizando ESP32-C6, teclado matricial, LCD 16x2, MQTT e dashboard web.

## Funcionalidades

- Senha via teclado
- LCD 16x2
- LEDs de status
- Comunicação MQTT
- Dashboard Web
- Liberação remota
- Alteração remota de senha

## Tecnologias

- ESP32-C6
- Arduino IDE
- Python Flask
- MQTT
- HiveMQ
